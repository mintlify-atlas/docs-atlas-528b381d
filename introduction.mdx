# Welcome to Beaver Builder

Beaver Builder is the page builder and website builder that agencies and developers rely on. Create pixel-perfect websites with our visual editor - no coding required. Front-end editing, flexible templates, and enterprise-grade performance.

## Why Choose Beaver Builder?

Beaver Builder is a flexible drag and drop page builder that works on the front end of your WordPress website. Whether you're a beginner or a professional, you'll love taking control of your website. Stop writing HTML or wrestling with confusing shortcodes. With Beaver Builder, building beautiful, professional WordPress pages is as easy as dragging and dropping.

## Key Features

### Content Modules

Beaver Builder comes loaded with flexible content modules to build stunning layouts:

- **Box** - Create flexbox and CSS grid layouts with presets
- **Photo, Button, Button Group** - Essential visual elements
- **Callout & Call to Action** - Drive engagement
- **Heading, Icon, Text Editor** - Core content elements
- **Menu, Number Counter** - Interactive components
- **HTML, Audio, Video, Sidebar** - Rich media and custom code

### Layout System

- Full-width, column-based layouts
- Lightweight and semantic markup for maximum performance
- Photo, color, and video row backgrounds
- Add your own CSS classes and IDs
- Works with Pages, Posts, and Custom Post Types

### Performance Optimized

<Note>
Beaver Builder only loads the assets needed for a given layout or landing page. Unlike many page builders that load excessive scripts and stylesheets on every page, Beaver Builder is designed for speed.
</Note>

### Theme Compatibility

Beaver Builder was designed to work with almost any WordPress theme. Try Beaver Builder on your existing website today, or consider using Beaver Builder's framework theme (available in Pro and Agency packages) for your next project.

## Developer Friendly

- Build your own custom modules
- Use WordPress Widgets and shortcodes
- Extend with hooks and filters
- Clean, semantic markup
