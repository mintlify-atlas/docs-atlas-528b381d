title: Premium & Integration Modules

---

Premium and integration modules extend Beaver Builder with advanced features and connections to third-party services.

## WooCommerce

Display products or categories from your WooCommerce store.

**Key Features:**
- Product display
- Category browsing
- Add to cart integration
- Product filtering
- Custom layouts
- Price display
- Product images

**Requirements:** WooCommerce plugin installed and activated

**Category:** WooCommerce

---

## BigCommerce Products

Displays the BigCommerce product gallery.

**Key Features:**
- BigCommerce integration
- Product display
- Shopping cart integration
- Product information
- Custom styling

**Requirements:** BigCommerce WordPress plugin

**Category:** BigCommerce

---

## North Commerce

Display products or categories from your North Commerce store.

**Key Features:**
- North Commerce integration
- Product listings
- Category display
- Store integration
- Custom layouts

**Requirements:** North Commerce platform connection

**Category:** E-commerce

---

## Content Slider

Displays multiple slides with an optional heading and call to action.

**Key Features:**
- Custom slide content
- Background images/videos
- Heading and text
- Call to action buttons
- Transition effects
- Autoplay options
- Navigation controls
- Multiple layout options

**Category:** Media

---

## Map

Display a Google map.

**Key Features:**
- Google Maps integration
- Custom markers
- Address input
- Map styling
- Zoom controls
- Custom map types
- Responsive sizing

**Requirements:** Google Maps API key

**Category:** Media

---

## Countdown

Render a Countdown module.

**Key Features:**
- Date and time target
- Days, hours, minutes, seconds
- Custom labels
- Number formatting
- Circle or inline style
- Action after countdown
- Timezone support

**Category:** Info

---

## Reusable Block

Display a WordPress Pattern.

**Key Features:**
- WordPress Block Pattern integration
- Reusable content
- Block editor compatibility
- Global updates
- Content synchronization

**Requirements:** WordPress Block Editor (Gutenberg)

**Category:** WordPress Integration

---

## ACF Block

Display an ACF block.

**Key Features:**
- Advanced Custom Fields integration
- Custom block types
- ACF field display
- Block template support
- Dynamic content

**Requirements:** Advanced Custom Fields Pro plugin

**Category:** WordPress Integration

---

## Widget

Display a WordPress widget.

**Key Features:**
- Any registered widget
- Widget customization
- Theme widget support
- Plugin widget support
- Custom styling

**Usage:**
Embed any WordPress widget into your Beaver Builder layout. Useful for maintaining compatibility with widget-based plugins.

**Category:** WordPress Widgets
