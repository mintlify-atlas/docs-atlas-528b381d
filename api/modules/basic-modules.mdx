---
title: Basic Modules
description: Core modules for fundamental content and design elements in Beaver Builder
---

Basic modules provide the essential building blocks for creating content in Beaver Builder. These modules handle common elements like buttons, text, images, and media.

## Button

A simple call to action button.

**Key Features:**
- Customizable styles and colors
- Gradient and advanced styling options
- Icon support
- Responsive alignment
- Link settings with multiple action types

**Category:** Basic

---

## Button Group

Renders a series of call to action buttons.

**Key Features:**
- Multiple buttons in a single module
- Flexible alignment options
- Consistent styling across buttons
- Responsive layout control

**Category:** Basic

---

## Heading

Display a title/page heading.

**Key Features:**
- HTML heading tags (H1-H6)
- Typography controls
- Text alignment
- Link options
- Custom styling

**Category:** Basic

---

## Text Editor

A WYSIWYG text editor.

**Key Features:**
- Rich text formatting
- WordPress editor integration
- HTML editing capability
- Media embedding
- Custom CSS classes

**Category:** Basic

---

## HTML

Display raw HTML code.

**Key Features:**
- Direct HTML input
- JavaScript support
- Custom code embedding
- No WYSIWYG filtering

**Category:** Basic

---

## Photo

Upload a photo or display one from the media library.

**Key Features:**
- Image upload or media library selection
- Crop options
- Link settings
- Caption support
- Responsive sizing
- Alignment controls

**Category:** Basic

---

## Video

Render a WordPress or embedable video.

**Key Features:**
- YouTube, Vimeo, and other embed support
- WordPress native video
- Custom video files
- Autoplay and loop options
- Responsive sizing

**Category:** Basic

---

## Audio

Render a WordPress audio shortcode.

**Key Features:**
- Audio file upload
- Media library integration
- WordPress audio player
- Playback controls

**Category:** Basic

---

## Icon

Display an icon and optional title.

**Key Features:**
- Font Awesome icon library
- Custom icon upload
- Size and color controls
- Optional text/link
- Animation effects

**Category:** Media

---

## Icon Group

Display a group of linked Font Awesome icons.

**Key Features:**
- Multiple icons in a single module
- Individual link settings
- Alignment options
- Consistent styling
- Social media integration

**Category:** Media

---

## Separator

A divider line to separate content.

**Key Features:**
- Horizontal line divider
- Style options (solid, dashed, dotted)
- Width and height controls
- Color customization
- Spacing controls

**Category:** Basic

---

## List

A simple list of items.

**Key Features:**
- Ordered or unordered lists
- Custom icons or numbers
- Individual item styling
- Spacing controls
- Nested list support

**Category:** Basic
