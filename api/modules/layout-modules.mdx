---
title: Layout Modules
description: Structural modules for organizing and arranging content in Beaver Builder
---

Layout modules help you structure and organize content on your page. These modules provide containers and interactive elements for better content organization.

## Box

A simple layout container.

**Key Features:**
- Flexible content container
- Accepts all module types
- Background options
- Border and padding controls
- Custom width and height
- Advanced styling options

**Usage:**
The Box module serves as a versatile container for grouping content. Use it to create card layouts, feature boxes, or any custom container design.

**Category:** Box

---

## Accordion

Display a collapsible accordion of items.

**Key Features:**
- Multiple accordion items
- Expand/collapse functionality
- Open by default options
- Custom icons
- Individual item styling
- Smooth animations

**Usage:**
Perfect for FAQs, nested content, or any scenario where you want to save vertical space while providing access to detailed information.

**Category:** Layout

---

## Tabs

Display a collection of tabbed content.

**Key Features:**
- Horizontal or vertical tabs
- Multiple tab items
- Active tab styling
- Responsive layout
- Custom icons
- Link options

**Usage:**
Organize related content into tabs for easy navigation. Great for product features, service descriptions, or content categorization.

**Category:** Layout

---

## Sidebar

Display a WordPress sidebar that has been registered by the current theme.

**Key Features:**
- Theme sidebar integration
- Widget area display
- Responsive positioning
- Custom styling options

**Usage:**
Integrate theme sidebars directly into your Beaver Builder layouts. Useful for maintaining widget areas within page builder designs.

**Category:** Layout

---

## Menu

Renders a WordPress menu.

**Key Features:**
- WordPress menu integration
- Mobile responsive
- Dropdown support
- Custom styling
- Multiple layout options
- Hamburger menu option

**Usage:**
Add navigation menus anywhere in your layout. Create custom headers, footer menus, or in-page navigation.

**Category:** Actions

---

## Pricing Table

A simple pricing table generator.

**Key Features:**
- Multiple pricing columns
- Feature lists
- Highlight options
- Button integration
- Responsive stacking
- Custom styling

**Usage:**
Create pricing comparisons for products or services. Perfect for subscription plans, service tiers, or product comparisons.

**Category:** Layout
