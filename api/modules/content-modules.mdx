---
title: Content Modules
description: Dynamic content and interactive modules for rich user experiences in Beaver Builder
---

Content modules provide dynamic and interactive elements for engaging your visitors. These include galleries, forms, posts, and other rich content types.

## Gallery

Display multiple photos in a gallery view.

**Key Features:**
- Multiple layout options (grid, masonry, collage)
- Lightbox integration
- Click actions
- Image captions
- Responsive columns
- Custom spacing

**Category:** Media

---

## Slideshow

Display multiple photos in a slideshow view.

**Key Features:**
- Image transitions
- Autoplay options
- Navigation controls
- Captions and overlays
- Ken Burns effect
- Speed controls

**Category:** Media

---
